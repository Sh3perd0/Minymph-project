Minymph-project

Pokemon-inspired java project. Source code: src/minymph_project/

How to run it? 
Eclipse version

- Clone the repository
- Extract it
- Open Eclipse
- Go to File
- "Open Projects from File System..."
- Import the folder
- Click on Main.java and run it (CTRL+F11)

Command-line version
- Clone the repo
- Extract it
- cd [yourFolderPath]/
